# 📱 아이폰(iPhone) 사용 가이드

요한일서 성경통독 앱을 아이폰에서 사용하는 방법을 안내합니다.

## 방법 1: GitHub Pages 이용 (가장 쉬운 방법) ⭐

### 단계
1. GitHub에서 이 저장소를 Fork 또는 업로드합니다
2. **Settings** → **Pages**에서 GitHub Pages 활성화
3. 생성된 URL(예: `https://your-username.github.io/bible-chatbot/index-mobile.html`)을 Safari에서 엽니다
4. 하단 공유 버튼(□↑) 클릭 → **홈 화면에 추가**
5. 이름 설정 후 **추가** 클릭

이제 홈 화면에서 앱처럼 바로 실행할 수 있습니다!

---

## 방법 2: iCloud Drive 이용

### 단계
1. 컴퓨터에서 `index-mobile.html` 파일을 iCloud Drive에 업로드
2. 아이폰에서 **파일** 앱 열기
3. **iCloud Drive** 선택
4. `index-mobile.html` 파일 찾아서 탭
5. Safari에서 열리면 사용 가능

### 홈 화면에 추가하기
1. Safari에서 파일이 열린 상태에서
2. 하단 공유 버튼(□↑) 클릭
3. **홈 화면에 추가** 선택
4. 이름 입력 (예: "요한일서")
5. **추가** 클릭

---

## 방법 3: Google Drive 이용

### 단계
1. 컴퓨터에서 `index-mobile.html` 파일을 Google Drive에 업로드
2. 아이폰에서 **Google Drive** 앱 열기
3. `index-mobile.html` 파일 찾기
4. 파일 옆 **⋮** (점 3개) 탭
5. **다음으로 열기** 선택
6. **Safari로 복사** 선택
7. Safari에서 열리면 사용 가능

---

## 방법 4: AirDrop 이용

### 단계
1. Mac에서 `index-mobile.html` 파일 선택
2. **공유** → **AirDrop** 선택
3. 아이폰으로 전송
4. 아이폰에서 받은 파일 탭
5. **Safari로 열기** 선택

---

## 방법 5: 이메일 첨부

### 단계
1. 컴퓨터에서 자신에게 이메일 전송 (`index-mobile.html` 첨부)
2. 아이폰에서 메일 앱 열기
3. 첨부 파일 다운로드
4. 첨부 파일 탭 → Safari에서 열기

---

## ✅ 작동 확인 체크리스트

### 파일이 제대로 열렸는지 확인:
- [ ] 보라색 그라데이션 헤더가 보임
- [ ] "요한일서 성경통독" 타이틀이 보임
- [ ] 7개의 카드(1장~5장, 요한일서 설명, 나의 묵상)가 보임
- [ ] 카드를 터치하면 내용이 열림

### 문제 해결

#### 카드가 터치되지 않는 경우:
1. Safari 설정에서 JavaScript가 활성화되어 있는지 확인
   - **설정** → **Safari** → **고급** → **JavaScript** 켜기
2. 페이지 새로고침 (⟳ 버튼)
3. Safari를 완전히 종료 후 재시작

#### 화면이 깨져 보이는 경우:
1. Safari에서 열어야 합니다 (Chrome, Firefox는 제한적 지원)
2. iOS 13 이상 권장

#### 로컬 스토리지(묵상 저장) 안 되는 경우:
1. Safari 설정 확인:
   - **설정** → **Safari** → **개인 정보 보호 및 보안**
   - **모든 쿠키 차단** 끄기
   - **추적 방지** 끄기 (선택사항)
